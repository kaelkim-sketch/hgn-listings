─────────────────────────────────────────────

─────────────────────────────────────────────
assets/img/

  about/          → 상담사 소개 섹션 이미지
  listings/
    1/            → 1번 현장 이미지
    2/            → 2번 현장 이미지
    3/            → 3번 현장 이미지

─────────────────────────────────────────────
gallery/ — 프로필 갤러리 (썸네일 4장)

3.jpg    현장 상담 사진      권장 비율: 1:1

about/ — 상담사 소개 섹션
─────────────────────────────────────────────

listings/1/, listings/2/, listings/3/ — 현장 카드
─────────────────────────────────────────────
