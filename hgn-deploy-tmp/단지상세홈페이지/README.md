# evan kang - 개인 작업 공간

## 폴더 구조

```
_personal/evan kang/
├── README.md              # 이 파일
├── _archive/              # 완료/만료된 문서 보관
├── meeting-notes/         # 회의록
│   ├── GUIDE.md           # 회의록 작성 가이드
│   └── YYYYMMDD_회의명.md
└── projects/              # 프로젝트 기획서
    ├── GUIDE.md           # 기획서 작성 가이드
    └── 프로젝트명.md
```

## 네이밍 규칙

| 문서 유형 | 형식 | 예시 |
|-----------|------|------|
| 회의록 | `YYYYMMDD_회의명.md` | `20260601_호강노노_CX어드민_기획리뷰.md` |
| 프로젝트 기획서 | `프로젝트명.md` | `hogangnono-cx-admin-v2.md` |

## 문서 상태 관리

기획서 상단 **상태** 필드로 진행 상태를 관리합니다.

| 상태 | 의미 |
|------|------|
| `기획중` | 작성 및 검토 진행 중 |
| `확정` | 개발 착수 가능한 상태 |
| `진행중` | 개발/실행 중 |
| `완료` | 출시 또는 종료 |
| `보류` | 중단 또는 재검토 대기 |
