이 폴더(assets)에 아래 "이름"으로 이미지를 넣으면 홈페이지에 자동 반영됩니다.
확장자는 .jpg / .png / .webp 아무거나 가능. 넣지 않으면 기존(샘플) 이미지가 그대로 쓰입니다.

  메인배경       → 첫 화면 풀스크린 배경 (메인 조감도 / 투시도)   [권장: 가로 1920px 이상]
  위치도         → 입지 섹션 + 확대 팝업의 위치도 / 배치도
  공유썸네일      → 카카오톡·페이스북 공유 시 보이는 미리보기 그림  [권장: 1200 x 630]
  오시는길약도    → '오시는 길' 섹션의 약도 이미지 (선택)

예시) 이 폴더에  메인배경.jpg ,  위치도.png ,  공유썸네일.png  를 넣으면 됩니다.

※ 그 외 추가 조감도·평면도 등을 더 넣고 싶으면, 파일을 build\img\ 에 두고
  build\index.html 의 해당 섹션에 <img src="img/파일명"> 한 줄을 추가하세요.
