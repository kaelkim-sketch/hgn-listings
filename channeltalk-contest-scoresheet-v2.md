### ② 시안 완성도　　/ 30점

> 카피·정보 구조·CTA의 명확성

**채점 기준**

| 점수 | 기준 |
|:---:|------|
| 10점 | 탁월 — 즉각 전달, 업계 최상위 완성도 |
| 8~9점 | 우수 — 기준 충족, 소폭 보완 가능 |
| 6~7점 | 보통 — 대체로 충족, 개선 여지 있음 |
| 4~5점 | 미흡 — 일부 미충족, 보완 필요 |
| 2~3점 | 불량 — 전반 미충족 |
| 0~1점 | 해당 없음 / 현저히 부족 |

**세부 평가**

| 항목 | 기준 | 점수 |
|-----|------|:---:|
| 카피 명확성 | 핵심 메시지가 3초 안에 전달되는가 | / 10 |
| 정보 구조 | 입지·가격·혜택이 위계 있게 배치되었는가 | / 10 |
| CTA 명확성 | 행동 유도 문구·버튼이 직관적인가 | / 10 |
| **소계** | | **/ 30** |

### ③ 랜딩 완성도　　/ 30점

> 채널톡 시안 → 랜딩페이지 연결성·완성도 전반
> 채점 기준은 ② 시안 완성도와 동일

**세부 평가**

| 항목 | 기준 | 점수 |
|-----|------|:---:|
| 메시지 일관성 | 채널톡 → 랜딩페이지의 톤·정보가 자연스럽게 이어지는가 | / 10 |
| 디자인 완성도 | 랜딩페이지 비주얼·레이아웃·가독성이 완성도 있게 구성되었는가 | / 10 |
| 전환 설계 | 문의·상담 유도 흐름이 자연스럽고 이탈 없이 설계되었는가 | / 10 |
| **소계** | | **/ 30** |

### ④ 현장 난이도 보정　　/ 10점

> 같은 CTR이라도 어려운 시장에서 달성한 성과를 더 높이 평가
> **카테고리 기준 CTR 대비 달성 배율**로 산정 — 기준 미달 시 0점

**카테고리 기준 CTR** (직방·호갱노노 집행 데이터 기반, 매 회차 공표)

| 카테고리 | 기준 CTR |
|---------|:-------:|
| 서울·수도권 아파트 신규 | 3.5% |
| 광역시 아파트 / 수도권 오피스텔·기타 | 2.5% |
| 지방 아파트 신규 | 1.5% |
| 전국 오피스텔·생활숙박·지산·상가 | 1.0% |
| 미분양 보유 현장 (지역·상품 무관) | 0.8% |

**달성 배율 → 점수** 　`달성 배율 = 실측 CTR ÷ 카테고리 기준 CTR`

| 달성 배율 | 점수 |
|:--------:|:---:|
| 3.0배 이상 | 10점 |
| 2.0배 ~ 2.9배 | 7점 |
| 1.5배 ~ 1.9배 | 4점 |
| 1.0배 ~ 1.4배 | 2점 |
| 1.0배 미만 | 0점 |
